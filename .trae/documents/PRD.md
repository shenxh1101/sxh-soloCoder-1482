## 1. 产品概述

工地工人考勤与工资管理系统，帮助工地管理者高效记录工人出勤情况、自动计算工资并管理借支记录。

- 解决人工记工容易出错、工资计算繁琐、借支管理混乱等痛点
- 面向工地包工头、现场管理人员，提供轻量化的考勤与薪资管理工具

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 管理员 | 本地登录 | 全部功能：工人管理、考勤记录、工资计算、借支管理、数据导出 |

### 2.2 功能模块

1. **仪表盘首页**：数据概览、今日出勤统计、本月工资概览、快捷操作入口
2. **工人管理**：工人信息增删改查、按工种分组查看、日薪/包活单价设置
3. **考勤记录**：按工种批量点卯、出勤/请假/旷工标记、日历视图查看
4. **工资结算**：月度工资自动计算、点工按天/包活按平米、借支自动扣除
5. **借支管理**：借支记录登记、借支历史查询、剩余工资实时显示

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|---------------------|
| 仪表盘 | 统计卡片 | 显示总工人数、今日出勤、本月应发、借支总额 |
| 仪表盘 | 快捷操作 | 快速跳转到考勤打卡、添加工人、登记借支 |
| 工人管理 | 工人列表 | 按工种分组展示，显示姓名、工种、日薪、包活单价 |
| 工人管理 | 添加工人 | 表单录入：姓名、工种、日薪、包活单价、入职日期 |
| 考勤打卡 | 工种分组点卯 | 按工种显示工人列表，批量勾选出勤、请假、旷工 |
| 考勤打卡 | 日期选择 | 选择考勤日期，支持补录历史考勤 |
| 考勤记录 | 日历视图 | 按日历展示每日出勤情况，点击查看详情 |
| 工资结算 | 工资列表 | 按月显示每个工人的出勤天数、工作量、工资、借支、实发 |
| 工资结算 | 工资明细 | 查看单个工人的工资计算明细 |
| 借支管理 | 借支登记 | 登记工人借支金额、日期、备注 |
| 借支管理 | 借支历史 | 查看所有借支记录，支持按工人筛选 |

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深靛蓝色 (#1e3a8a) —— 稳重、专业，适合工程管理场景
- **辅助色**：琥珀色 (#d97706) —— 用于强调和警示
- **成功色**：翠绿色 (#059669) —— 出勤标记
- **中性色**：石板灰系列 —— 文字、边框、背景
- **按钮风格**：圆角矩形 (rounded-lg)，悬停有微缩放和阴影变化
- **字体**：标题使用思源黑体 Bold，正文使用思源黑体 Regular
- **布局风格**：左侧导航栏 + 右侧内容区，卡片式布局
- **图标风格**：使用 Lucide React 线性图标

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|-----------|-------------|-------------|
| 仪表盘 | 统计卡片 | 4列网格，图标+数字，渐变背景色，轻微阴影 |
| 仪表盘 | 快捷操作 | 圆角按钮组，图标+文字，悬停动效 |
| 工人管理 | 分组列表 | 工种折叠面板，工人卡片，操作按钮 |
| 考勤打卡 | 点卯区域 | 工人工号+姓名网格，三态切换按钮(出勤/请假/旷工) |
| 工资结算 | 工资表格 | 斑马纹表格，合计行高亮，金额右对齐 |
| 借支管理 | 借支表单 | 下拉选择工人，数字输入框，日期选择器 |

### 4.3 响应式

- 桌面端优先设计 (≥1024px)
- 平板端 (768-1023px)：导航栏折叠为图标，表格横向滚动
- 移动端 (<768px)：底部Tab导航，卡片式列表替代表格

## 5. 数据导出

- 支持导出 CSV 格式，兼容 Excel 打开
- 导出内容：工人花名册、月度考勤表、工资结算表、借支记录表
- 文件名格式：`工地名称_报表类型_年月.csv`
